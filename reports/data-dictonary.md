# Data Dictionary

## Project

**Bluestock Fintech - Mutual Fund Analytics Capstone**

This data dictionary describes the key fields used in the Mutual Fund Analytics project. It provides the column name, data type, business definition, and source dataset.

| Column            | Data Type | Business Definition                                                                                        | Source                    |
| ----------------- | --------- | ---------------------------------------------------------------------------------------------------------- | ------------------------- |
| amfi_code         | Integer   | Unique identifier assigned to each mutual fund scheme by AMFI. Used to link data across multiple datasets. | fund_master.csv           |
| scheme_name       | String    | Official name of the mutual fund scheme.                                                                   | fund_master.csv           |
| fund_house        | String    | Asset Management Company (AMC) responsible for managing the mutual fund scheme.                            | fund_master.csv           |
| category          | String    | Broad classification of the mutual fund (e.g., Equity, Debt, Hybrid).                                      | fund_master.csv           |
| sub_category      | String    | More specific classification within the fund category (e.g., Large Cap, Mid Cap, Liquid Fund).             | fund_master.csv           |
| plan              | String    | Type of mutual fund plan such as Direct or Regular.                                                        | fund_master.csv           |
| risk_category     | String    | Risk level assigned to the mutual fund scheme (e.g., Low, Moderate, High, Very High).                      | fund_master.csv           |
| date              | Date      | Date on which the Net Asset Value (NAV) was recorded.                                                      | nav_history.csv           |
| nav               | Float     | Net Asset Value (NAV) per unit of the mutual fund on the specified date.                                   | nav_history.csv           |
| transaction_date  | Date      | Date on which an investor performed a transaction.                                                         | investor_transactions.csv |
| transaction_type  | String    | Type of investor transaction such as SIP, Lumpsum, or Redemption.                                          | investor_transactions.csv |
| amount_inr        | Integer   | Amount invested or redeemed by the investor in Indian Rupees (INR).                                        | investor_transactions.csv |
| investor_id       | String    | Unique identifier assigned to each investor.                                                               | investor_transactions.csv |
| state             | String    | State in which the investor is located.                                                                    | investor_transactions.csv |
| city              | String    | City in which the investor is located.                                                                     | investor_transactions.csv |
| payment_mode      | String    | Mode of payment used for the transaction (e.g., UPI, Net Banking, Cheque).                                 | investor_transactions.csv |
| kyc_status        | String    | Status of the investor's Know Your Customer (KYC) verification.                                            | investor_transactions.csv |
| expense_ratio_pct | Float     | Annual percentage charged by the fund for managing investor assets.                                        | scheme_performance.csv    |
| return_1yr_pct    | Float     | Annual return generated by the mutual fund over the last one year.                                         | scheme_performance.csv    |
| return_3yr_pct    | Float     | Annualized return generated by the mutual fund over the last three years.                                  | scheme_performance.csv    |
| return_5yr_pct    | Float     | Annualized return generated by the mutual fund over the last five years.                                   | scheme_performance.csv    |
| benchmark_3yr_pct | Float     | Three-year return of the benchmark index used to compare the fund's performance.                           | scheme_performance.csv    |
| sharpe_ratio      | Float     | Risk-adjusted performance measure indicating return earned per unit of risk.                               | scheme_performance.csv    |
| alpha             | Float     | Excess return of the mutual fund compared to its benchmark after adjusting for market risk.                | scheme_performance.csv    |
| beta              | Float     | Measure of the fund's volatility compared to the overall market.                                           | scheme_performance.csv    |

## Notes

* **AMFI Code** is the primary identifier used to establish relationships between datasets.
* **NAV (Net Asset Value)** represents the market value of one unit of a mutual fund.
* All date fields are converted to the **datetime** data type during the data cleaning process.
* Financial metrics such as returns, NAV, and expense ratio are validated before loading into the SQLite database.
* This data dictionary serves as a reference for developers, analysts, and stakeholders working with the Mutual Fund Analytics database.
